Recursos

Alguns dos recursos que compõem a ELISA

Projetos de caixas

Para cada uma dessas caixas a Elisa calcula configuração normal ou isobárica, fazendo com que a quantidade total de caixas se extenda para 30 tipos diferentes.

Maior banco de dados de subwoofers do mundo.

Possui o maior banco de dados de mids; midbass; woofers e subwoofers do planeta Terra. Este banco é atualizado constantemente.

Crossovers; bitola de cabos; etc...

Dentro do software Elisa você encontra também uma seção para calcular crossovers passivos de 6 a 24dB/8º; bitola de cabos para falantes e amplificadores e várias outras funções.

Cálculo do valor de custo / revenda de caixas.

Não perca dinheiro! Elisa calcula o valor de custo da caixa, incluindo todos os insumos e custos da empresa e lhe oferece o valor correto para que seja aplicada a margem de lucro líquido.
